## Function
The autonomous boat contains 4 main functions:
* Drive straight stably in the specified direction

* Boat cover x 1; 
* Battery case x 1; 
* 1.5v Battery x 4; 
* Breadboard x 1; 
* LSM303 3 Axis Digital Compass sensor;

* Eletric motor x2 ;
